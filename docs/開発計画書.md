**目標:** 既存機能を壊さずに、使いやすさの土台を整える

- [ ] 既存コードの調査・構造把握

- [ ] ボタン配色の変更
- [ ] ボタンのプロパティ編集画面の修正
- [ ] コード整理（重複処理のクラス化）

- [ ] UIコンポーネントの統一・整理

- [ ] 付箋機能の実装
