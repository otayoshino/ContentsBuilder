## 2026-03-04（続き②）

### 作業内容

#### 5. 付箋：ドラッグで任意サイズのボックスを作成する機能を追加

**概要**
- 「付箋」パネルボタンを押下するとページ上で描画モードに移行
- ページ上でドラッグして任意サイズの矩形を作成
- マウスアップ後に付箋設定ダイアログを表示
- ダイアログ保存後、指定位置・サイズに付箋ノートを配置

**追加CSS（`mock/index.html`）**
| セレクタ | 用途 |
|---|---|
| `.sticky-note` | 配置済み付箋ボックス |
| `.sticky-content` | 付箋テキストエリア |
| `.sticky-corner` | 右下の折り返しコーナー |
| `.sticky-delete-btn` | ホバー時に表示される削除ボタン |
| `.sticky-draw-preview` | ドラッグ中の点線プレビュー矩形 |
| `.page.drawing-mode` | 描画モード時のクロスヘアカーソル |

**追加JS関数**
| 関数名 | 役割 |
|---|---|
| `activateStickyMode()` | 描画モードを有効化 |
| `deactivateStickyMode()` | 描画モードを無効化 |
| `onPageMouseDown(e)` | ページのmousedownイベント（描画開始） |
| `onPageMouseMove(e)` | ページのmousemoveイベント（プレビュー更新） |
| `onPageMouseUp(e)` | ページのmouseupイベント（矩形確定・ダイアログ表示） |
| `openStickySettingsDialog()` | 付箋設定ダイアログを動的生成して表示 |
| `confirmStickyNote()` | ダイアログ保存時に付箋をページへ配置 |
| `makeDraggable(el)` | 付箋を移動可能にする |
| `selectStickyNote(el)` | 付箋クリックで選択状態を切り替える |
| `deleteStickyNote(el)` | 付箋を削除する |

**操作フロー**
1. 「付箋」ボタン押下 → ドラッグモード（ステータスバーにメッセージ表示）
2. ページ上でドラッグ → 点線プレビュー矩形が表示される
3. マウスアップ → 設定ダイアログが開く（内容・背景色・フォント）
4. 「保存」 → 付箋が指定位置に配置（ドラッグで移動可能、ホバーで削除ボタン表示）
5. Escキー → 描画モードをキャンセル

#### 7. 複数選択時のライブアップデート改善（デルタ方式）

**概要**
- 複数オブジェクト選択中にサイドバーのX/Y/W/H値を変更した際、各オブジェクトの相対位置・個別サイズを維持したまま変更を適用する

**改修前の問題**
- 複数選択時にX/Y/W/Hを絶対値で全オブジェクトに適用していたため、全オブジェクトが同一位置・同一サイズに揃ってしまっていた

**修正内容**
- `applyLiveUpdate()` を以下のルールで再実装：
  - 単一選択：絶対値適用（変更なし）
  - 複数選択：X/Y/W/H はデルタ（差分）を適用し、カラーは絶対値適用
- `applyLiveUpdate._prevX/_prevY/_prevW/_prevH` を関数プロパティとして状態管理
- `openAnnotationSettingsDialog()` と `closeDialog()` でデルタ基準値をリセット

#### 8. 複数選択中の Alt+ドラッグで全オブジェクトをコピー

**概要**
- 複数オブジェクト選択中に、選択内のいずれかを Alt+ドラッグすると全選択オブジェクトをまとめてコピーできる機能を追加

**改修前の動作**
- 複数選択中に Alt+ドラッグしてもドラッグ対象の1件のみコピーされていた

**修正内容**
- `makeDraggable()` の `mousedown` ハンドラ先頭に複数選択チェックを追加
- 複数選択中（2件以上）かつ選択内オブジェクトで Alt を押した場合：
  - `getSelectedObjects()` で全選択オブジェクトを取得してそれぞれクローン
  - 各クローンに新 ID を割り振り、クリックハンドラ・リサイズを再設定（`reinitElement()`）
  - 開始時の各オブジェクト位置を記録し、マウス移動デルタを全クローンに一括適用
  - Shift+Alt ドラッグで水平/垂直軸固定にも対応
- 単体選択 or 選択外オブジェクトの Alt+ドラッグは従来通り1件のみコピー

| 状況 | Alt+ドラッグの挙動 |
|---|---|
| 複数選択中・選択内をドラッグ | 全選択オブジェクトをコピーし、相対位置を維持したまま移動 |
| 単体選択 or 選択外オブジェクト | そのオブジェクトのみコピー（従来通り） |
